Chart Monitor – Datenschutzerklärung

Deutsch / English

Entwickler: Boner Timm Peter Emanuel (Stock Monitor) | www.stock-monitor.ch

Datenerhebung

Chart Monitor erhebt und speichert keine persönlichen Daten. Die App erfordert keine Registrierung und kein Benutzerkonto.

Netzwerkzugriff

Die App ruft live Aktienkurse, Wechselkurse und Fundamentaldaten von Yahoo Finance (api.finance.yahoo.com, query1.finance.yahoo.com, query2.finance.yahoo.com) ab. Dabei werden nur die angefragten Ticker-Symbole übertragen – keine Nutzeridentifikation.

Optional prüft die App auf neue Versionen über die GitHub Releases API (api.github.com). Auch dabei werden keine persönlichen Daten übertragen.

Lokale Speicherung

Die App speichert ausschliesslich Benutzereinstellungen lokal auf dem Gerät (SharedPreferences): zuletzt gesuchte Aktien, Chart-Einstellungen, Watchlist-Einträge inkl. Preisalarme, bestätigte Datenschutzerklärung sowie FX-Rechner-Einstellungen. Es werden keine Daten an externe Server übertragen.

Hintergrundaktivität

Wenn Kursalarme (Stop-Loss / Zielkurs) konfiguriert sind, prüft die App periodisch im Hintergrund die aktuellen Kurse über Yahoo Finance. Auch dabei werden keine persönlichen Daten übertragen.

Drittanbieter

  • Yahoo Finance – Kursdaten (keine personenbezogenen Daten)
  • GitHub – Update-Prüfung (keine personenbezogenen Daten)

Chart Monitor – Privacy Policy

Deutsch / English

Developer: Boner Timm Peter Emanuel (Stock Monitor) | www.stock-monitor.ch

Data Collection

Chart Monitor does not collect or store personal data. No registration or user account is required.

Network Access

The app fetches live stock prices, exchange rates, and fundamental data from Yahoo Finance (api.finance.yahoo.com, query1.finance.yahoo.com, query2.finance.yahoo.com). Only the requested ticker symbols are transmitted – no user identification.

Optionally, the app checks for new versions via the GitHub Releases API (api.github.com). No personal data is transmitted in this process.

Local Storage

The app only stores user preferences locally on the device (SharedPreferences): last searched stocks, chart settings, watchlist entries including price alerts, confirmed privacy policy, and FX calculator settings. No data is sent to external servers.

Background Activity

If price alerts (Stop-Loss / Target Price) are configured, the app periodically checks current prices in the background via Yahoo Finance. No personal data is transmitted.

Third Parties

  • Yahoo Finance – Price data (no personal data)
  • GitHub – Version check (no personal data)

## [1.0.0] – 2026-06-10

### Added
- Initial release
- Stock chart display with time ranges: 1D, 1W, 1M, 3M, 6M, 1Y, 5Y, Max
- Compare two stocks in the same chart (percentage-normalized view)
- Search by ISIN, ticker symbol, or company name (via Yahoo Finance)
- Automatic DE/EN language detection with matching number format
- Price display with current price, currency, and change in percent
- Info dialog: Stock Monitor reference, license (GPL-3.0), privacy policy, contact, donate (Twint & PayPal)
- Light/Dark mode follows system setting
- Supports Android 5.0+ (API 21), optimized for 6–7" screens
